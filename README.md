##  注意！！！

+ 由于校园网路由限制，为实现前后端分离项目的正常联通，需要设置前后端服务器不在同一子局域网下

+ 推荐配置如下：

    | 可选 |          前端          |          后端           |
    | :--: | :--------------------: | :---------------------: |
    |  1   | BUAA-WiFi（局域网ip）  | BUAA-Mobile（局域网ip） |
    |  2   | BUAA-WiFi（localhost） | BUAA-Mobile（局域网ip） |
